# 権限説明文

## storage

授業選択、課題候補、締切情報、提出状態、最終更新時刻をユーザーのブラウザ内に保存するために使用します。

## activeTab

ユーザーが開いているLETUSページから、授業一覧を読み取るために使用します。

## scripting

ユーザー操作時に、開いているLETUSページ内の授業リンクを抽出するために使用します。

## notifications

更新完了や、締切が近い課題を通知するために使用します。

通知はユーザーのブラウザ内に保存された課題情報をもとに表示されます。

## host_permissions: https://letus.ed.tus.ac.jp/*

ユーザーが選択したLETUS上の授業ページおよび課題ページから、課題名、締切、提出状態を読み取るために使用します。

対象ドメインはLETUSに限定しています。

## リモートコード

本拡張機能はリモートコードを実行しません。

拡張機能に含まれるJavaScriptファイルのみを実行します。
