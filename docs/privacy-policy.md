# プライバシーポリシー

## LETUS Task Watcher について

LETUS Task Watcher は、LETUS上の課題締切と提出状況を確認しやすく整理するためのブラウザ拡張機能です。

この拡張機能は、ユーザーがLETUS上の課題を管理しやすくすることを目的としています。

## 取得する情報

LETUS Task Watcher は、LETUS上から以下の情報を取得する場合があります。

- 課題名
- コース名
- 締切
- 提出状況
- 課題URL
- 対象コース情報

また、拡張機能の利用に必要な設定情報として、以下の情報を保存します。

- 対象コース設定
- 非表示にした課題
- 前回更新時刻
- 通知済み状態
- 更新状態

## 情報の利用目的

取得した情報は、以下の目的でのみ使用されます。

- 課題一覧の表示
- 24時間以内の課題表示
- 今週の課題表示
- 締切前通知
- 更新完了通知
- 対象コース設定
- 非表示リスト管理
- 保存データ管理

## 外部送信について

LETUS Task Watcher は、取得した課題情報や設定情報を外部サーバーへ送信しません。

保存される情報は、ユーザーのブラウザ内の `chrome.storage.local` に保存されます。

## 第三者提供について

LETUS Task Watcher は、取得した情報を第三者へ提供しません。

## 広告利用について

LETUS Task Watcher は、取得した情報を広告目的で利用しません。

## リモートコードについて

LETUS Task Watcher は、外部サーバーから実行コードを取得して実行することはありません。  
拡張機能に含まれるコードのみを使用します。

## データの削除

ユーザーは、ダッシュボードのデータ管理機能から、保存された課題データや設定情報を削除できます。

削除できる主なデータは以下の通りです。

- 課題データ
- 更新状態
- 対象コース設定
- 非表示リスト
- すべての保存データ

## 公式サービスとの関係

LETUS Task Watcher は、LETUS公式の拡張機能ではありません。  
LETUS Task Watcher は、LETUS上の課題情報を整理して表示するための非公式補助ツールです。
