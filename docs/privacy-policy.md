# LETUS Task Watcher プライバシーポリシー

最終更新日: 2026年6月26日

LETUS Task Watcher（以下、「本拡張機能」）は、LETUS上の授業ページから課題締切および提出状況を整理するための非公式ブラウザ拡張機能です。

本拡張機能は、東京理科大学またはLETUS公式のサービスではありません。

## 1. 収集・保存する情報

本拡張機能は、ユーザーのブラウザ内に以下の情報を保存します。

- ユーザーが選択した授業名
- 授業ページURL
- 課題名
- 課題ページURL
- 課題の締切情報
- 課題の提出状態
- 読み取り対象として選択した授業設定
- 最終更新時刻

これらの情報は、ユーザーの端末内のブラウザストレージに保存されます。

## 2. 収集しない情報

本拡張機能は、以下の情報を収集・保存しません。

- LETUSのID
- LETUSのパスワード
- 大学アカウントの認証情報
- Cookie
- セッション情報
- 成績情報の収集を目的としたデータ
- ユーザーが提出したファイルの内容

## 3. 外部送信について

本拡張機能は、読み取った授業情報・課題情報・締切情報・提出状態を外部サーバーへ送信しません。

本拡張機能の処理は、原則としてユーザーのブラウザ内で完結します。

## 4. LETUSへのアクセスについて

本拡張機能は、ユーザーがLETUSにログインしている状態のブラウザ上で動作します。

本拡張機能は、ユーザーが選択した授業ページおよび課題ページにアクセスし、課題名、締切、提出状態を読み取ります。

本拡張機能は、LETUSへのログイン処理を代行しません。

## 5. 通知について

本拡張機能が通知機能を使用する場合、通知はユーザーのブラウザ内に保存された課題情報をもとに表示されます。

通知情報は外部サーバーへ送信されません。

## 6. データの削除

ユーザーは、ブラウザの拡張機能設定から本拡張機能を削除することで、本拡張機能が保存したデータを削除できます。

また、本拡張機能の画面内にある削除機能により、保存済みの授業情報・課題情報を削除できます。

## 7. 第三者提供

本拡張機能は、保存したデータを第三者に提供しません。

## 8. 免責事項

本拡張機能は、LETUSの表示内容をもとに課題締切を整理しますが、すべての締切・提出状態の正確性を保証するものではありません。

最終的な課題締切および提出状況は、必ずLETUS公式画面で確認してください。
