# LETUS Task Watcher

LETUS Task Watcher は、LETUS上の課題締切と提出状況を確認しやすく整理するためのブラウザ拡張機能です。

ポップアップでは、24時間以内の課題や今週の課題をコンパクトに確認できます。  
詳しい課題一覧、対象コースの選択、非表示にした課題の管理、保存データのリセットなどは、ダッシュボード画面から行えます。

## 主な機能

- 24時間以内の課題を優先表示
- 明日まで、今週、それ以降の課題分類
- 対象コースの選択
- 対象コースの一括ON/OFF
- 開始前、提出済み、期限切れの課題表示
- 不要な課題の非表示
- 非表示にした課題の個別復元
- 更新中の進捗表示
- 更新完了時の通知
- 締切前通知
- 保存データのリセット

## ポップアップ

ポップアップでは、普段すぐに確認したい情報だけを表示します。

- 24時間以内の課題
- 今週の課題数
- 対象コース数
- 前回更新時刻
- 今すぐ更新
- ダッシュボードへの移動

## ダッシュボード

ダッシュボードでは、課題や設定を詳しく管理できます。

- 課題一覧の詳細確認
- 対象コースの選択
- 対象コースの一括ON/OFF
- 非表示にした課題の管理
- 提出済み課題の確認
- 期限切れ課題の確認
- 保存データの管理

## 非表示機能

不要な課題は、ダッシュボードから非表示にできます。

非表示にした課題は完全に削除されるわけではなく、非表示リストに保存されます。  
誤って非表示にした場合でも、非表示リストから個別に戻すことができます。

## 通知機能

LETUS Task Watcher は、以下の場合に通知を表示します。

- 更新が完了したとき
- 24時間以内の未提出課題があるとき
- 締切が近い課題があるとき

## 保存されるデータ

この拡張機能は、以下の情報をユーザーの端末内に保存します。

- 課題名
- コース名
- 締切
- 提出状況
- 課題URL
- 対象コース設定
- 非表示にした課題
- 前回更新時刻
- 通知済み状態
- 更新状態

取得したデータを外部サーバーへ送信することはありません。

## 対応ブラウザ

- Google Chrome
- Microsoft Edge

## 注意

LETUS Task Watcher は、LETUS公式の拡張機能ではありません。  
この拡張機能は、LETUS上の課題情報を整理して表示するための非公式補助ツールです。
