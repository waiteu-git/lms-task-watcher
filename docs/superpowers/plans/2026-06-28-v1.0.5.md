# v1.0.5 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** v1.0.5 として、サブスクリプション関連UIをすべて除去したうえで、changelog.html（更新通知ページ）と初回オンボーディングバナーを実装し、ビルドして Chrome Web Store 提出用 ZIP を作成する。

**Architecture:** `changelog.html` は `public/` に置く静的HTMLファイル。`onInstalled` の `reason === 'update'` 時に background から新規タブで開く。オンボーディングバナーは `OnboardingBanner.tsx` コンポーネントをポップアップ最上部に表示し、`chrome.storage.local` の `onboardingCompleted` フラグで初回のみ表示を制御する。スキャン正常完了時にフラグを立て永久非表示にする。

**Tech Stack:** React 19 + TypeScript, Vite, Vitest, Chrome Extensions MV3

## Global Constraints

- バニラJSを追加しない（TypeScript のみ）
- `public/` に置いたファイルはビルド設定変更なしで `dist/` にコピーされる
- テスト実行: `pnpm exec vitest run`
- ビルド: `pnpm build`
- `chrome.storage.local` のキーはすべて `src/background/storageKeys.ts` に定数として定義する
- コメントは WHY が非自明な場合のみ記述する
- **v1.0.x ではサブスクリプション関連のUI・ロジックを一切ユーザーに見せない**（コードファイル自体は削除せず、App.tsx からの参照・レンダリングのみ除去する）

import { ONBOARDING_COMPLETED_KEY } from '../background/storageKeys'

export async function getOnboardingCompleted(): Promise<boolean> {
  const result = await chrome.storage.local.get(ONBOARDING_COMPLETED_KEY) as {
    onboardingCompleted?: boolean
  }
  return result.onboardingCompleted === true
}

export async function setOnboardingCompleted(): Promise<void> {
  await chrome.storage.local.set({ [ONBOARDING_COMPLETED_KEY]: true })
}

```tsx
import type { Course } from '../core/types'

type Props = {
  courses: Course[]
  lastRefreshAt: string | null
}

type Step = 1 | 2 | 3

function resolveStep(courses: Course[], lastRefreshAt: string | null): Step {
  if (courses.length === 0) return 1
  if (courses.filter((c) => c.enabled).length === 0) return 2
  return 3
}

export function OnboardingBanner({ courses, lastRefreshAt }: Props) {
  if (lastRefreshAt !== null) return null

  const step = resolveStep(courses, lastRefreshAt)

  function openLetus() {
    void chrome.tabs.create({ url: 'https://letus.ed.tus.ac.jp' })
  }

  function openDashboard() {
    void chrome.tabs.create({ url: chrome.runtime.getURL('index.html#dashboard') })
  }

  return (
    <div className="onboardingBanner">
      <div className="onboardingStep">
        {step === 1 && (
          <>
            <p className="onboardingStepLabel">ステップ 1 / 3</p>
            <p className="onboardingText">
              まず LETUS にアクセスしてください。コースページを開くと自動で登録されます。
            </p>
            <button type="button" className="onboardingBtn" onClick={openLetus}>
              LETUS を開く →
            </button>
          </>
        )}
        {step === 2 && (
          <>
            <p className="onboardingStepLabel">ステップ 2 / 3</p>
            <p className="onboardingText">
              ダッシュボードを開いて、追跡したいコースにチェックを入れてください。
            </p>
            <button type="button" className="onboardingBtn" onClick={openDashboard}>
              ダッシュボードを開く →
            </button>
          </>
        )}
        {step === 3 && (
          <>
            <p className="onboardingStepLabel">ステップ 3 / 3</p>
            <p className="onboardingText">
              準備完了！「今すぐ更新」を押して課題を取得してください。
            </p>
          </>
        )}
      </div>
    </div>
  )
}
```
