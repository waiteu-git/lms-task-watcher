# マイページ機能 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Webサイト（lms.waiteu.dev）にログイン・マイページ機能を追加し、Stripeカスタマーポータル経由で支払い方法を更新できるようにする。

**Architecture:** 既存の`POST /api/auth/login`はそのまま利用し、新規`POST /api/subscription/billing-portal`（要認証）がStripeの`stripe.billingPortal.sessions.create()`でセッションURLを発行する。Webサイトは`landing/login.html`（ログイン）と`landing/mypage.html`（状態表示・支払い管理ボタン）を素のHTML/JSで新規追加し、JWTを`localStorage`に保存する。

**Tech Stack:** 既存のExpress API（`api/`、better-sqlite3、Stripe SDK）、Jest + supertest（`cd api && npx jest`で実行）、素のHTML/CSS/JS（`landing/`、ビルド不要）。

## Global Constraints

- 統計機能はスコープ外（LETUS提出日時パース機能が未実装のため）
- 支払い方法更新はStripeカスタマーポータルへのリダイレクト方式（独自UIは作らない）
- `localStorage`のキー名: `authToken`（JWT文字列）・`authTokenExpiresAt`（ISO文字列）。これがWebサイト側で初めてlocalStorageを使うため、この命名を正とする
- `mypage.html`は`authToken`が無ければ即座に`login.html`へリダイレクトする
- 新規ページは`landing/register.html`と同じCSS変数命名（`--accent`, `--text-dark`, `--text-mid`, `--border`, `--bg-pale`, `--border-pale`）・ヘッダー/フッター構造に合わせる
- テストは`cd api && npx jest`で実行する（リポジトリルートからの`vitest run`はapi/testsを対象にしないこと）
- Stripeダッシュボードでcustomer portalが有効化されている前提（ユーザー側の作業、このプラン実行前に完了していること）

```js
// POST /api/subscription/billing-portal
router.post('/billing-portal', requireAuth, async (req, res) => {
  const sub = db.prepare(
    'SELECT stripe_customer_id FROM subscriptions WHERE user_id = ?'
  ).get(req.userId)

  if (!sub || !sub.stripe_customer_id) {
    return res.status(404).json({ error: 'Stripe顧客情報が見つかりません' })
  }

  try {
    const session = await stripe.billingPortal.sessions.create({
      customer: sub.stripe_customer_id,
      return_url: 'https://lms.waiteu.dev/mypage.html',
    })

    return res.json({ url: session.url })
  } catch (err) {
    console.error(err)
    return res.status(500).json({ error: 'カスタマーポータルセッションの作成に失敗しました' })
  }
})
```

```html
<script>
  const API_BASE_URL = 'https://api.waiteu.dev'
  const contentEl = document.getElementById('content')
  const errorEl = document.getElementById('error')

  function clearAuthAndRedirect() {
    localStorage.removeItem('authToken')
    localStorage.removeItem('authTokenExpiresAt')
    window.location.href = 'login.html'
  }

  const token = localStorage.getItem('authToken')
  if (!token) {
    clearAuthAndRedirect()
  }

  async function loadStatus() {
    try {
      const res = await fetch(`${API_BASE_URL}/api/subscription/status`, {
        headers: { Authorization: `Bearer ${token}` },
      })

      if (res.status === 401) {
        clearAuthAndRedirect()
        return
      }

      const data = await res.json()

      if (!res.ok) {
        errorEl.textContent = data.error || 'ステータスの取得に失敗しました'
        errorEl.style.display = 'block'
        return
      }

      renderStatus(data)
    } catch {
      errorEl.textContent = 'サーバーに接続できませんでした'
      errorEl.style.display = 'block'
    }
  }

  function renderStatus(data) {
    if (data.status === 'active') {
      const periodEndText = data.currentPeriodEnd
        ? new Date(data.currentPeriodEnd).toLocaleDateString('ja-JP')
        : '不明'

      contentEl.innerHTML = `
        <div class="card">
          <p class="card-label">次回請求日</p>
          <p class="card-value">${periodEndText}</p>
          <button id="portal-btn" type="button">支払い方法を管理</button>
        </div>
      `

      document.getElementById('portal-btn').addEventListener('click', openBillingPortal)
    } else {
      contentEl.innerHTML = `
        <div class="card status-inactive">
          <p class="card-label">サブスクリプション状態</p>
          <p class="card-value">現在サブスクリプションが有効ではありません</p>
          <a class="register-link" href="register.html">サブスクリプションに登録する →</a>
        </div>
      `
    }
  }

  async function openBillingPortal() {
    const btn = document.getElementById('portal-btn')
    btn.disabled = true
    btn.textContent = '処理中...'

    try {
      const res = await fetch(`${API_BASE_URL}/api/subscription/billing-portal`, {
        method: 'POST',
        headers: { Authorization: `Bearer ${token}` },
      })
      const data = await res.json()

      if (!res.ok || !data.url) {
        errorEl.textContent = data.error || 'カスタマーポータルの取得に失敗しました'
        errorEl.style.display = 'block'
        btn.disabled = false
        btn.textContent = '支払い方法を管理'
        return
      }

      window.location.href = data.url
    } catch {
      errorEl.textContent = 'サーバーに接続できませんでした'
      errorEl.style.display = 'block'
      btn.disabled = false
      btn.textContent = '支払い方法を管理'
    }
  }

  document.getElementById('logout-btn').addEventListener('click', clearAuthAndRedirect)

  if (token) {
    void loadStatus()
  }
</script>
</body>
</html>
```
