# Chrome Web Store 掲載文案

## 拡張機能名

LETUS Task Watcher

## 短い説明

LETUSの授業ページから課題締切と提出状況を整理する学生向け拡張機能です。

## 詳細説明

LETUS Task Watcherは、LETUS上の授業ページから課題締切・提出状況を読み取り、ブラウザ内で整理する学生向けの非公式拡張機能です。

## 主な機能

- LETUSの授業一覧を読み取り
- 読み込む授業をユーザーが選択
- 選択した授業だけ課題候補を取得
- 課題締切を一覧表示
- 24時間以内の課題を優先表示
- 今後の課題を締切が近い順に表示
- 開始前・提出済み・期限切れを分類表示
- 課題カードをクリックしてLETUSの該当課題ページを開く
- 読み取り中の進捗表示
- ユーザーの端末内にデータを保存

## プライバシー

本拡張機能は、LETUSのID・パスワードを収集しません。

授業情報・課題情報・締切情報・提出状態は、ユーザーのブラウザ内に保存されます。

本拡張機能は、これらの情報を外部サーバーへ送信しません。

## 注意事項

本拡張機能は、東京理科大学またはLETUS公式のサービスではありません。

課題締切や提出状況は、必ずLETUS公式画面でも確認してください。

LETUSの画面構成変更により、一部の課題情報を正しく読み取れない場合があります。
