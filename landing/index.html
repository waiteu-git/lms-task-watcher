できること

LETUSにログインするだけで、あとは自動で動きます

課題を自動収集

LETUSへのログインセッションを使い、登録済みコースの課題リンクと締切日を自動で取得します。手動コピーは不要です。

締切前に通知

締切の3時間前と1時間前にデスクトップ通知でお知らせ。各課題につき1回だけ通知するので重複しません。

ダッシュボードで一覧管理

課題を「24時間以内」「明日」「今週」「それ以降」のグループに分類して表示。提出状況も一目で確認できます。

バックグラウンド自動更新

2時間以上データが古くなると自動で再スキャン。「今すぐ更新」での手動更新も可能。

データはすべてブラウザ内

収集したデータはブラウザのローカルストレージにのみ保存。外部サーバーへの送信は一切行いません。

かんたん導入

インストール後、LETUSにログインして拡張機能アイコンをクリックするだけ。設定不要で即使えます。

使い方

4ステップで完了

インストール

Chrome Web StoreからLETUS Task Watcherを追加します。

LETUSにログイン

普段通りLETUSにログインします。追加の操作は不要です。

コースを選択

ダッシュボードを開き、追跡したいコースにチェックを入れます。

スキャン開始

「今すぐ更新」をタップ。課題が締切順に一覧表示されます。

プライバシー

あなたのデータを大切に扱います

収集・送信するデータについて

  • ログインID・パスワードは一切読み取りません
  • 収集データはブラウザのローカルストレージにのみ保存
  • 外部サーバーへのデータ送信は行いません
  • ダッシュボードからいつでもデータを削除できます
  • アンインストールすると保存データもすべて削除されます
詳細はプライバシーポリシーをご覧ください。
